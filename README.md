# Olá, eu sou o Mateus! 

## 🚀 Backend Developer | .NET Specialist

Sou um desenvolvedor focado em construir sistemas robustos e escaláveis utilizando o ecossistema **.NET**. Meu trabalho é pautado pela entrega de software de alta qualidade, aplicando **Clean Architecture**, **Domain-Driven Design (DDD)** e os princípios **SOLID**.

🛠 **No que venho trabalhando:**

* **Ecossistema .NET:** Desenvolvimento de APIs RESTful com **ASP.NET Core**, **EF Core** e **Identity**.
* **Qualidade & Testes:** Implementação de **TDD** com **xUnit** para garantir códigos resilientes.
* **DevOps & Infra:** Containerização com **Docker**, versionamento via **Git** e automação com **GitHub Actions**.
* **Metodologias:** Experiência prática com **Scrum** e **Kanban** para entregas ágeis e incrementais.

Atualmente, estou focado no projeto **MyRecipeBook**, onde aplico padrões avançados de desenvolvimento backend e segurança.
