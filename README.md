# Olá, eu sou o Mateus

## Backend Software Engineer | .NET Specialist

Sou um desenvolvedor focado em construir sistemas robustos, escaláveis e de alta performance utilizando o ecossistema **.NET**. Minha atuação é pautada pela entrega de software de alta qualidade, aplicando **Clean Architecture**, **Domain-Driven Design (DDD)** e os princípios **SOLID**.

---

### Expertise & Foco Técnico

* **Arquitetura & Design:** Desenvolvimento de APIs RESTful com **ASP.NET Core**, garantindo segurança e performance com **Identity**, **JWT** e **OAuth2**.
* **Persistência de Dados:** Experiência avançada com **Entity Framework Core**, otimização de queries e modelagem de dados em **SQL Server** e **MySQL**.
* **Qualidade de Software:** Defensor da cultura de testes, aplicando **TDD** com **xUnit**, **Moq** e **FluentAssertions**.
* **DevOps & Infra:** Containerização com **Docker**, versionamento via **Git** e automação de fluxos de CI/CD com **GitHub Actions**.

> [!IMPORTANT]
> **Projeto em Destaque:** Atualmente focado no **MyRecipeBook**, onde aplico padrões avançados de desenvolvimento backend, criptografia e regras de negócio complexas.
